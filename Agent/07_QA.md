### SK-QA-02: Plagiarism Check
- ตรวจสอบการลอกเลียน / self-plagiarism
- ตรวจ similarity score
- แนะนำ paraphrase ถ้า similarity สูง
```python
print("🔍 Plagiarism Check")
threshold = 20  # percent
for chapter in chapters:
    score = check_similarity(chapter)
    status = "✅" if score < threshold else "❌"
    print(f"  {status} {chapter}: {score}%")
```

### SK-QA-03: Data Validation
- ตรวจข้อมูลวิจัยถูกต้อง ครบถ้วน
- ตรวจ calculations ตามกรอบวิจัย
- Cross-check ตัวเลขระหว่าง tables/figures
- Output → `Test/data_validation_report.md`

### SK-QA-04: Format Validation
- ตรวจรูปแบบตาม template มหาวิทยาลัย/journal
- ตรวจ: margins, fonts, spacing, numbering, headers
- Output → `Test/format_check.md`

### SK-QA-05: Final Checklist
- ตรวจสอบก่อน submit ครั้งสุดท้าย
- Checklist:
  - [ ] สารบัญตรงกับเนื้อหา
  - [ ] เลขหน้าถูกต้อง
  - [ ] ภาพ/ตารางมี caption ครบ
  - [ ] References ครบทุกตัว
  - [ ] Abstract ไม่เกิน word limit
  - [ ] ไม่มี TODO/FIXME ค้าง
  - [ ] Spell check ผ่าน
