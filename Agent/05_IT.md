# 💻 IT Agent (ไอที)

> **Role:** จัดการระบบ, automation, environment, backup  
> **Script:** `Scripts/health_check.py`, `Scripts/backup.py`

---

## 🧠 Skills

### SK-IT-01: Environment Setup
- ติดตั้ง Python, dependencies, virtual environment
- จัดการ `Config/.env`, API keys
- ตรวจสอบ compatibility

### SK-IT-02: Backup & Version Control
- Auto backup ข้อมูลวิจัย → cloud ทุกวัน
- Git commits, branching strategy
- เก็บ old versions → `Archive/`
```python
print("💾 Backup Options")
print("1. Backup ทั้งโปรเจค")
print("2. Backup เฉพาะ Output/")
print("3. Backup เฉพาะ Data/")
choice = input("[1/2/3]: ")
```

### SK-IT-03: Data Pipeline
- Raw data (`Data/`) → cleaned → analyzed (`Workspace/experiments/`)
- Script: `Scripts/data_pipeline.py`
- รองรับ CSV, Excel, JSON

### SK-IT-04: Automation Scripts
- สร้างและดูแล automation scripts ทั้งหมดใน `Scripts/`
- Report generation อัตโนมัติ
- Citation formatting อัตโนมัติ

### SK-IT-05: System Monitoring
- ตรวจ API keys ยังใช้ได้, disk space, log errors
- Output → `Logs/system_health.md`
- Schedule: รายสัปดาห์

### SK-IT-06: Sandbox Management
- จัดการ `Workspace/Sandbox/` สำหรับทดสอบ agent
- ทดสอบ script ใหม่ก่อน deploy จริง
- Rollback ถ้ามีปัญหา

### SK-IT-07: Config Management
- จัดการ `Config/` — API keys, model settings, paths
- Security: API key rotation รายเดือน
- ไม่ commit secrets เข้า git
