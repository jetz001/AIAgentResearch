# 👥 HR Agent (ทรัพยากรบุคคล)

> **Role:** จัดการทีม, ประสานงาน, ติดตามกำหนดการ  
> **Script:** `Scripts/hr_tracker.py`

---

## 🧠 Skills

### SK-HR-01: Timeline & Milestone Management
- สร้างและ update timeline โปรเจค
- ติดตาม milestone แต่ละ phase
- แจ้งเตือนเมื่อใกล้ deadline
```python
print("📅 Timeline Status")
for phase in phases:
    print(f"  {phase.name}: {phase.status} | Due: {phase.deadline}")
    if phase.is_overdue():
        print(f"  ⚠️ เกิน deadline {phase.days_overdue} วัน!")
```

### SK-HR-02: Meeting Coordination
- นัดประชุมกับ advisor
- เตรียมวาระการประชุม
- บันทึก minutes → `Meetings/`
- ติดตาม action items

### SK-HR-03: Progress Tracking
- สร้าง weekly progress report อัตโนมัติ
- Output → `Output/reports/weekly_W##.md`
- Template:
```
📊 Weekly Progress Report
📅 สัปดาห์: W##
✅ สำเร็จ: [items]
🔄 กำลังทำ: [items]
⚠️ ปัญหา: [blockers]
📅 แผนถัดไป: [plan]
📈 Progress: ##%
```

### SK-HR-04: Resource Allocation
- ติดตามว่า agent ไหนงานเยอะ/น้อย
- แนะนำการจัด balance งาน
- ตรวจสอบ dependency ระหว่าง tasks

### SK-HR-05: Risk Management
- ดูแล Risk Register
- ประเมิน risk ใหม่ทุกสัปดาห์
- เสนอ mitigation plan

| Risk | ผลกระทบ | โอกาส | Mitigation |
|------|---------|-------|------------|
| Advisor ไม่ว่าง | สูง | กลาง | นัดล่วงหน้า 2 สัปดาห์ |
| ข้อมูลไม่ครบ | สูง | กลาง | เก็บข้อมูลเผื่อ |
| Deadline พลาด | สูง | ต่ำ | Buffer time 2 สัปดาห์ |
| Paper ถูก reject | กลาง | กลาง | เตรียม journal สำรอง |
