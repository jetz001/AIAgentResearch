**ภาคผนวก ค – รายละเอียดการคำนวณคาร์บอน**  

1. **การกำหนดขอบเขตการประเมิน**  
   - ขอบเขต หนึ่ง (Direct) : การปล่อยโดยตรงจากกิจกรรมขององค์กร  
   - ขอบเขต สอง (Indirect) : การปล่อยโดยอ้อมจากการใช้ไฟฟ้าและความร้อนที่ซื้อจากภายนอก  
   - ขอบเขต สาม (Indirect) : การปล่อยโดยอ้อมจากห่วงโซ่อุปทานและการจัดการของเสีย  

2. **สูตรการคำนวณ**  

   - ปริมาณการปล่อย ก๊าซเรือนกระจก ( กก. คาร์บอนเทียบเท่า )  

     \[
     \text{ค​ก​ร​} = \sum_{i=1}^{n}\left( \text{ปริมาณกิจกรรม}_{i}\times \text{ป​ฟ}_{i} \right)
     \]  

     โดยที่  
     * \(\text{ป​ฟ}_{i}\) คือ ค่าสัมประสิทธิ์ ป​ฟ ของกิจกรรมที่ \(i\)  
     * \(\text{ปริมาณกิจกรรม}_{i}\) คือ จำนวนหน่วยกิจกรรมที่วัดได้ (เช่น กิโลวัตต์‑ชั่วโมง หรือ กิโลกรัม )  

   - การแปลงเป็น กก. คาร์บอนเทียบเท่า  

     \[
     \text{ก​ก​ร​} = \text{ค​ก​ร​}\times \text{ค่าการแปลง}
     \]  

     ค่าการแปลงใช้ค่า คาร์บอนเทียบเท่าตามประเภทก๊าซ (เช่น คาร์บอนไดออกไซด์ = 1, มีเทน = 25  เป็นต้น)  

3. **ขั้นตอนการคำนวณ**  

   1. รวบรวมข้อมูลปริมาณกิจกรรมจากระบบบันทึกขององค์กร  
   2. นำค่าป​ฟที่อ้างอิงจากภาคผนวก ข มาคูณกับข้อมูลกิจกรรมตามสูตรข้อ 2  
   3. ยอดรวมของแต่ละกิจกรรมจะแสดงผลเป็น กก. คาร์บอนเทียบเท่า  
   4. สรุปผลรวมตามขอบเขต หนึ่ง – สาม เพื่อนำเสนอเป็นค่า คาร์บอนฟุตพริ้นท์ ขององค์กร  

4. **การตรวจสอบความถูกต้อง**  

   - ตรวจสอบความสอดคล้องของข้อมูลกิจกรรมกับบันทึกการใช้พลังงานจริง  
   - ตรวจสอบค่าป​ฟกับเอกสารอ้างอิงมาตรฐานอย่างเป็นระบบ  
   - ทำการสอบทานซ้ำโดยผู้ตรวจสอบอิสระก่อนนำผลสรุปไปใช้ในรายงาน
