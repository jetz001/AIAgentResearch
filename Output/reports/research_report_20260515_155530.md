### 1️⃣ วัตถุประสงค์ (Objective) & สรุปกระบวนการคิด (Thinking Process)

| วัตถุประสงค์หลัก | รายละเอียด |
|-------------------|------------|
| **รวบรวมข้อมูลพื้นฐานของโรงงาน** | ขนาด, กำลังการผลิต, แหล่งพลังงาน, วัตถุดิบ |
| **จัดทำแผนผังการไหลของกระบวนการผลิต** (Process Flow Diagram – PFD) | เพื่อใช้เป็นพื้นฐานสำหรับการคำนวณคาร์บอนฟุตพริ้นท์ (Scope 1‑3) ในวิทยานิพนธ์ |

#### กระบวนการคิด (🧠 Thinking Process)

1. **ทำความเข้าใจภาระงาน** – ต้องได้ข้อมูลเชิงปริมาณของโรงงานและแผนผังกระบวนการผลิตเป็น “single source of truth” สำหรับการวิเคราะห์ GHG.  
2. **ระบุ Skill ที่ต้องใช้** – เลือก SK‑RES‑01,‑03,‑04,‑07 (Shared Context),‑08,‑06,‑07 (Secondary) ตาม Matrix ของระบบ.  
3. **วางแผน 3 Phase**  
   - **Phase 1**: ค้นหาและคัดกรองวรรณกรรม → สกัดข้อมูลสำคัญ → บันทึกใน `Docs/literature_database.csv`, `Docs/selected_papers.md`, `Docs/literature_matrix.md`.  
   - **Phase 2**: Gap Analysis → ออกแบบเครื่องมือเก็บข้อมูล (แบบสอบถาม, เช็คลิสต์ LCA, Consent Form) → บันทึกใน `Workspace/Decision/gap_analysis.md`, `Docs/interview_questionnaire.md`, `Memory/Shared/Shared_Context.json`.  
   - **Phase 3**: ติดต่อและเก็บข้อมูลจากโรงงาน → สร้าง PFD ของแต่ละโรงงาน → วิเคราะห์ Emission (Scope 1‑3) → สรุปผลใน Shared Context ให้ Writer Agent ใช้ต่อ.  

> **ผลลัพธ์ของ Phase 1** – การสืบค้น **เสร็จสมบูรณ์** (ดูรายละเอียดด้านล่าง)

### 4️⃣ ข้อสังเกตและข้อเสนอแนะ (Observations & Recommendations)

| ข้อสังเกต | ข้อเสนอแนะ |
|-----------|-------------|
| **ข้อมูล Scope 3 ยังขาด** | ใช้ **GHG Protocol** เป็นแนวทางขอข้อมูลจากโรงงาน (การขนส่งวัตถุดิบ, การจัดการของเสีย) ‑ หรือใช้ค่าประมาณจาก **Ecoinvent / GaBi** เป็นตัวแทนในกรณีที่ไม่ได้รับข้อมูล |
| **ไม่มี PFD ที่เปิดเผย** ใน literature | ให้ทีมออกแบบ PFD ขั้นต้นจากข้อมูลเบื้องต้น (เช่น แรงดันเครื่องอบแห้ง, การตัดกระดาษ) แล้วปรับปรุงต่อเมื่อได้ข้อมูลจริงจากโรงงาน |
| **ความหลากหลายของแหล่งพลังงาน** (ไฟฟ้า/แก๊ส/พลังงานทดแทน) | ระบุ **Energy Mix** ในแบบสอบถามอย่างละเอียด (สัดส่วน % จากไฟฟ้า, แก๊ส, พลังงานแสงอาทิตย์) เพื่อคำนวน Emission Factor ที่แม่นยำ |
| **ความล่าช้าของการรับอนุญาต** | แนะนำให้ **เริ่มติดต่อโรงงาน 2‑3 สัปดาห์ล่วงหน้า** พร้อมแบบฟอร์ม Informed Consent ที่เตรียมไว้แล้ว |
| **การจัดการข้อมูล** | เก็บไฟล์ raw ใน `Data/raw_factory_data/` ด้วยการตั้งชื่อมาตรฐาน (`factoryID_YYYYMMDD.csv`) เพื่อให้ QA Agent ตรวจสอบได้ง่าย |
| **อ้างอิงมาตรฐาน** | ทุกขั้นตอนคำนวณ Emission ควรอ้างอิง **ISO 14064‑1**, **GHG Protocol**, **Thailand Carbon Footprint (TGO)** เพื่อความสอดคล้องกับข้อกำหนดของ Advisor |
| **การสื่อสารกับ Writer Agent** | ส่งไฟล์ `Shared_Context.json` พร้อมสรุป “Key Metrics” (size, capacity, EF, PFD link) เพื่อให้เขียนบทวิจัยต่อได้โดยไม่ต้องทำซ้ำการสืบค้น |
