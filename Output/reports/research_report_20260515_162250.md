1. **ทำความเข้าใจโจทย์** – แยกหัวข้อหลัก (LCA, Carbon Footprint, GHG accounting) และแยกขอบเขต (ปี 2019‑2024, ประเทศไทย + นานาชาติ) จากบันทึกของ Advisor.  
2. **กำหนด Keywords** – สร้างชุดคำค้นหลายชุด (อังกฤษ/ไทย) เพื่อให้ครอบคลุมทุกประเด็นที่ต้องการ.  
3. **เลือกเครื่องมือค้นหา** – ใช้ Scopus, Web of Science, Google Scholar, ฐานข้อมูลไทย (TCI, Thai Journals) พร้อมดึงรายงานภาครัฐ (สนพ., สภาอุตสาหกรรม).  
4. **ดำเนินการค้นหา** – รันสคริปต์ `phase1_literature.py` เพื่อดึง metadata (title, authors, year, DOI, citations) ลง `Docs/literature_database.csv`.  
5. **คัดกรองเบื้องต้น** – กรองตามปี ≤ 5 ปี, จำนวน citations ≥ 10, ความเกี่ยวข้องสูงกับ “corrugated cardboard”.  
6. **ตรวจสอบเต็มรูปแบบ** – อ่าน Abstract / Full‑text เพื่อตรวจสอบว่ามีการทำ LCA, กำหนด boundaries ตาม ISO 14040/44, รายงานค่า CO₂e หรือไม่.  
7. **สกัดข้อมูล** – สร้าง Literature Matrix (`Docs/literature_matrix.md`) ที่บันทึกรายละเอียดสำคัญของแต่ละ paper.  
8. **วิเคราะห์ Gap** – เปรียบเทียบว่ามีงาน LCA ของกล่องลูกฟูกในประเทศไทยหรือไม่, พบช่องว่างด้านข้อมูลเชิงปริมาณของ Thai case study.  
9. **บันทึก Context** – เก็บ Key Findings + citation ลง `Memory/Shared/Shared_Context.json` เพื่อให้ Writer Agent ใช้ต่อ.

## 4️⃣ ข้อสังเกตและข้อเสนอแนะ (Observations & Recommendations)

| ข้อสังเกต | คำอธิบาย |
|-----------|-----------|
| **1. ขาดข้อมูลเชิงปริมาณของไทย** | ไม่มีงานวิจัยหรือรายงานสาธารณะที่ให้ค่า CO₂e ของกระบอกลูกฟูกในประเทศไทย (kg CO₂e / หน่วย) หรืออัตราการรีไซเคิลที่ตรวจสอบได้ในช่วง 2019‑2024 |
| **2. งานสากลมีรายละเอียดครบ** | พบหลาย paper จากยุโรป, อเมริกา และจีนที่ทำ LCA เต็มรูปแบบ (ขอบเขต cradle‑to‑gate, cradle‑to‑grave) โดยใช้ SimaPro / OpenLCA |
| **3. แหล่งข้อมูลภาครัฐ** | รายงาน ESG ของบริษัทกระดาษไทยให้ข้อมูลเชิงคุณภาพ (เช่น การใช้พลังงาน, การรีไซเคิล) แต่ไม่แสดงค่า CO₂e อย่างเป็นตัวเลข |
| **4. ความท้าทายการเข้าถึง PDF** | บาง paper มีการจำกัดการเข้าถึง (pay‑walled) ทำให้ต้องขอความช่วยเหลือจาก IT Agent หรือใช้ institutional proxy เพื่อดึงไฟล์เต็ม |

### ข้อเสนอแนะเพื่อดำเนินการต่อ

1. **ขอความเห็นชอบจาก Orchestrator**  
   - ยืนยันการใช้ **gap analysis** ที่ได้เป็นพื้นฐานการเก็บข้อมูลภาคสนาม (Primary Data)  
   - ระบุว่าเราจำเป็นต้องทำ **case‑study data collection** จาก 3‑5 โรงงานในภาคกลาง‑ตะวันออก  

2. **ดำเนินการเก็บข้อมูลภาคสนาม**  
   - ใช้ **SK‑RES‑08 (Secondary Data Collection)** เพื่อรวบรวมรายงาน ESG, รายงานพลังงาน, ข้อมูลการขนส่งจากบริษัทที่ให้ข้อมูลตามข้อตกลงความลับ (NDA)  
   - จัดทำ **questionnaire** สำหรับผู้จัดการโรงงาน (วัตถุดิบ, ปริมาณการผลิต, ระยะทางขนส่ง, วิธีการกำจัดของเสีย)  

3. **เพิ่มการเข้าถึงเอกสารที่จำกัด**  
   - ติดต่อ **IT Agent** เพื่อขอบริการ VPN/Institutional Proxy หรือใช้บริการ **Unpaywall** เพื่อดึง PDF Open‑Access version  

4. **ทำขั้นตอน LCA เบื้องต้น**  
   - เลือก **OpenLCA** (ฟรี) หรือ **SimaPro trial** เพื่อทำ **pre‑LCA** ของกรณีศึกษา (ใช้ข้อมูลเบื้องต้นจาก literature และจาก questionnaire)  
   - ผลลัพธ์เบื้องต้นจะช่วยยืนยันว่ากระบวนการใดเป็น “hot‑spot” ของการปล่อย GHG  

5. **อัพเดท Gap Analysis**  
   - หลังจากเก็บข้อมูลภาคสนาม จะทำ **Gap‑Closing Analysis** เพื่อระบุว่าข้อมูลใดยังขาดและต้องทำการสำรวจต่อไป  

6. **สื่อสารกับ Writer Agent**  
   - ส่ง `Shared_Context.json` พร้อม **Literature Matrix** และ **Gap Analysis** เพื่อให้เขียนบทสรุปเชิงทฤษฎี (review) พร้อมคำแนะนำการทำ LCA ของประเทศไทย
